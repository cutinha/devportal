IOPS capability of plans
========================

When creating or updating an Aiven service, you can choose the plan, driving the number of resources available. The type of virtual machines created depends on the cloud provider chosen, and each one has different **IOPS** (Input/output Operations Per Second) limits. This reference provides the mapping between the common plans in Aiven for PostgreSQLÂ® and the related virtual machine types in the major clouds.

Business and premium
'''''''''''''''''''''

The cloud instance types in the Business and Premium plans are equal to the startup ones. The difference is the cluster size with 2 nodes for Business plans and 3 nodes high availability set for Premium plans.
